The goal is to minimize the amount of movement that needs to be done and thus minimize stress on the body What does the following mean?

  • No Movements Needed - No movement needed to type the characters because the finger already should be on the key
  • Finger Movements - Need to move the finger before typing a character
  • Same Finger Use - Same finger is used to type different letters ("lo" in "hello" with qwerty)

ax = df.sort_values(by='Finger Movements', ascending=False).plot(kind='bar', x='RESULTS', y='Finger Movements')
plt.xticks(rotation=45)
plt.show()

The chart above shows the required finger movements to type everything in our Python data set. We can see that QWERTY requires a lot more movement which means more work and strain on your fingers, joints, and wrists. Anything is better than QWERTY, but the versions of Colemak are clearly the winners here.
